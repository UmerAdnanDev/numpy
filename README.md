# NumPy Learning Repository

A comprehensive guide to NumPy, a fundamental Python library for numerical computing and data manipulation.

## Covered Topics

### Fundamentals
1. Introduction to NumPy
2. Array Creation and Initialization
3. Data Types
4. Basic Operations

### Array Operations
1. Attributes - Array properties and metadata
2. Indexing & Slicing - Accessing and extracting array elements
3. Alterations - Reshaping, resizing, and modifying arrays
4. Broadcasting - Operating on arrays of different shapes
5. Vectorization - Efficient element wise operations
